Counter Strike : Global Offensive Custom Maps

Project Details

• Language

>_ None, made using VALVe's Hammer World Editor

• Objective

>_ To create fun and difficult maps for the custom game mode (kz) in Counter Strike : Global Offensive

• Problems_Faced

>_ Had to learn how to use VALVe's hammer world editor, as well as learn a bit of C# for making materials.

>_ Had to learn how the hammer compiler works and understand how to optimize large open maps as well as avoid compiler errors.

• Knowlege_Gained

>_ Good knowledge of the hammer compiler.

>_ Learned map design and a great deal about QA. Many revisioned has to be made before a final version of the maps due to bugs in design that were previous overlooked. A good lesson that ties over into my programming projects

• Output

>_ The community had great responces to the final versions of the maps, kz_gitgud being the hardest, they were all enjoyed and played daily on the House Of Climb servers hosting the custom game type.

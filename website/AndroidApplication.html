Implicit Bias Tester (Android)

Project Details

• Language

>_ Java/XML

• Objective

>_ The main objective of this assignment was to learn and become comfortable with the Android API. This was my first time programming using Android's development enviroment, The majority of the program was finished in 6 days.

• Problems_Faced

>_ Having never used Android's API before, learning all the proper calls and the setup using XML files was a bit confusing, but in a day or 2 i had the hang of it.
>_ Getting references from objects between activities (Parcelable) and having session persistence (Serialization) was also an issue but was eventually overcome.

• Knowlege_Gained

>_ A strong knowledge of the Serializable and Parcelable interfaces, as well as gained experience working with prebuild frameworks, navigating documentation and choosing the appropriate method calls when multiple approaches are possible.

• Output

>_ For this project I recieved 190/200 possible points, the only points being lost were features i chose not to implement for the sake of smoothness and clarity.
